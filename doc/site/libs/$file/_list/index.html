    $file.list¶

    Description¶

    Énumère le contenu d'un dossier. La méthode retourne un tableau d'objets de type file.

    Syntaxe¶

    $file.list( directory [ , filter ] )

    Paramètres¶

    directory file / string
    Dossier dont on souhaite énumérer le contenu
    filter string
    Filtre des éléments à énumérer. Le filtre peut contenir des wildcards.

    Retour¶

    Tableau de file

    Exemple¶

    Exemple de fonction récursive qui énumère tout le contenu d'un répertoire:

    function listContent(file) {
    if ($file.getAttributes(file).isDirectory) {
        for (var f : $file.list(file)) {
            listContent(f);
        }
    }
    else {
        $logger.info(file);
    }
}
